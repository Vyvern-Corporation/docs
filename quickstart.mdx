description: "Start the backend and make your first authenticated request"
---

Follow this guide to run the backend locally and make your first API call.

<Steps>
<Step title="Start the backend">

```bash
cd backend
npm install
npm run dev
```

The server starts on `http://localhost:5000` by default.

</Step>

<Step title="Configure auth settings">

Create a `.env` file in `backend/` and set at least a JWT secret:

```bash
JWT_SECRET=replace-me
JWT_EXPIRES_IN=7d
FRONTEND_URL=http://localhost:3000
```

Restart the server after updating `.env`.

</Step>

<Step title="Register or log in">

Use `POST /api/auth/register` or `POST /api/auth/login` to get a token.

```bash
curl -s -X POST http://localhost:5000/api/auth/login \
  -H 'Content-Type: application/json' \
  -d '{"email":"admin@example.com","password":"example-pass"}'
```

Copy the `token` value from the response.

</Step>

<Step title="Create an organization">

Use the token to create your first org.

```bash
curl -s -X POST http://localhost:5000/api/organizations \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "organizationName":"Acme Co",
    "userCount":120,
    "hrmPlatform":"Workday",
    "solutionTypes":["email-phishing"],
    "hasSocialPresence":true
  }'
```

</Step>
</Steps>
