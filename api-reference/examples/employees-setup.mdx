## Request

```bash
curl -s -X POST http://localhost:5000/api/employees/setup \
  -H 'Content-Type: application/json' \
  -d '{
    "token":"<invite-token>",
    "password":"example-pass",
    "consentData": {
      "agreed": true,
      "signature": "Riley Nguyen",
      "ipAddress": "203.0.113.10",
      "userAgent": "Mozilla/5.0"
    }
  }'
```
