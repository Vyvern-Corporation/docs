---
title: 'Create an organization'
description: 'Example request and response for POST /api/organizations.'
---

## Request

```bash
curl -s -X POST http://localhost:5000/api/organizations \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "organizationName":"Acme Co",
    "userCount":120,
    "hrmPlatform":"Workday",
    "solutionTypes":["email-phishing","insider-risk"],
    "hasSocialPresence":true,
    "websiteUrl":"https://acme.example",
    "authorizedPlatforms":["gmail","slack"]
  }'
```
