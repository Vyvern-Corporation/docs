---
title: 'Update organization branding'
description: 'Example request and response for PUT /api/organizations/:id/settings.'
---

## Request

```bash
curl -s -X PUT http://localhost:5000/api/organizations/org_123/settings \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "name":"Acme Co",
    "description":"Security awareness training for Acme",
    "primaryColor":"#0EA5E9",
    "secondaryColor":"#111827",
    "websiteUrl":"https://acme.example"
  }'
```

## Response

```json
{
  "message": "Organization settings updated successfully",
  "organization": {
    "id": "org_123",
    "name": "Acme Co",
    "description": "Security awareness training for Acme",
    "logo": null,
    "logoCloudflareId": null,
    "primaryColor": "#0EA5E9",
    "secondaryColor": "#111827",
    "websiteUrl": "https://acme.example",
    "updatedAt": "2026-02-19T18:00:00.000Z",
    "logoUrl": null
  }
}
```
